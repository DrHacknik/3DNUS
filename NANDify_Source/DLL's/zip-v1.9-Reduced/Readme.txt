DotNetZip Reduced Library, Developer's Kit package, v1.9 packed Sat-08-06-2011-215945.06.   This is the reduced version of the DotNetZip library.  It includes the classes in the Ionic.Zip namespace as well as the classes in the Ionic.Zlib namespace.  The reduced library differs from the full library in that it lacks the ability to save self-Extracting archives (aka SFX files), and is much smaller than the full library.
